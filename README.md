# JaipurCircle - Jaipur's Premier Deals Platform

## About

JaipurCircle is Jaipur's most trusted deal discovery platform, connecting thousands of happy customers with verified local merchants. Discover exclusive deals and deep discounts on dining, shopping, services, and experiences across the Pink City.

## Features

- **Smart Deal Discovery**: Browse 1,500+ active deals from 800+ verified merchants
- **JAICoin Rewards**: Earn points on every purchase and interaction
- **Location-Based**: Find deals near you across all major Jaipur localities  
- **Merchant Dashboard**: Complete portal for businesses to manage deals and analytics
- **Gamification**: Daily rewards, challenges, and a competitive leaderboard
- **Referral Program**: Earn by inviting friends and referring merchants

## Technology Stack

This project is built with modern web technologies:

- React 18 with TypeScript
- Vite for blazing-fast development
- Tailwind CSS for responsive design
- Supabase for backend and authentication
- React Query for data management
- Shadcn/ui component library

## Getting Started

Clone the repository and install dependencies. Requires Node.js & npm - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Step 3: Install dependencies
npm i

# Step 4: Start the development server
npm run dev
```

## Development

```bash
npm run dev    # Start development server
npm run build  # Build for production
npm run preview # Preview production build
```

## Deployment

Deploy to production using your preferred hosting platform:

```bash
npm run build
```

The build output will be in the `dist` directory, ready to deploy to Vercel, Netlify, or any static hosting platform.
